# Histórias de Usuário

## Histórias de Usuário – Jogador

**Story 1**  
Como jogador de simulação, eu quero visualizar a velocidade do veículo em um painel físico para que eu tenha uma experiência de condução mais imersiva.

**Story 2**  
Como jogador de simulação, eu quero visualizar a rotação do motor em um painel físico para que eu possa acompanhar o desempenho do veículo de forma mais realista.

**Story 3**  
Como jogador de simulação, eu quero visualizar a marcha engatada em um display físico para que a condução se torne mais intuitiva e realista.

**Story 4**  
Como jogador de simulação, eu quero saber o estado de funcionamento do sistema para que eu possa utilizá-lo com confiança.

**Story 5**  
Como jogador de simulação, eu quero utilizar o sistema no Xbox Series para que eu possa utilizar o dispositivo na plataforma que possuo.

**Story 6**  
Como jogador de simulação, eu quero utilizar o sistema no PlayStation 4 para que eu possa utilizar o dispositivo na plataforma que possuo.

**Story 7**  
Como jogador de simulação, eu quero utilizar o sistema no PlayStation 5 para que eu possa utilizar o dispositivo na plataforma que possuo.

**Story 8**  
Como jogador de simulação, eu quero utilizar o sistema no PC para que eu possa utilizar o dispositivo na plataforma que possuo.

**Story 9**  
Como jogador de simulação, eu quero sentir vibrações ao passar sobre a faixa zebrada para que eu perceba fisicamente quando o veículo sai da trajetória ideal.

**Story 10**  
Como jogador de simulação, eu quero sentir uma vibração ao dar a partida no veículo para que o início da corrida seja mais realista.

## Histórias de Usuário – Sistema

**Story 1**  
Como sistema, eu quero me conectar automaticamente à rede local para que o usuário não precise realizar configurações manuais.

**Story 2**  
Como sistema, eu quero estabelecer comunicação com o jogo via UDP para que os dados de telemetria sejam recebidos continuamente.

**Story 3**  
Como sistema, eu quero interpretar os dados de telemetria recebidos para que eles possam ser utilizados no controle dos atuadores.

**Story 4**  
Como sistema, eu quero identificar automaticamente a conexão e desconexão de módulos para que o funcionamento seja contínuo e flexível.

**Story 5**  
Como sistema, eu quero iniciar automaticamente após ser ligado para que o usuário possa utilizá-lo sem etapas adicionais.
